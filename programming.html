Neural Network Predictor

Skills:Python, Neural Networks, Predictive Modeling, Databases, Machine Learning

During my internship at New York Life, one of my projects aims to use data about virtual sales communications patterns to predict an agent's sales performance. After some experiments, I learned that using neural network and the keras API would be an optimal choice for the research problem. Thus, I normalzed data, removed outliers and implemented a deep learning model that significantly improves the accuracy of prediction. Furthermore, by looking at the statistical importance of the different factors that affect sales performance, I've been able to identify the most important communication patterns and virtual activity patterns that tell the difference in sales performance.

Art Critique Chat

Skills:Computer Networking, Client-Server Architecture, Java, Multi-threading programming

During my computer network class, I worked on creating a "Live Chat" room that allowed users to exchange text and images through messages. The chat simulates the experience of going to an exhibition by randomly displaying pictures by automatically switching to random image every 15 seconds , allowing users to comment and read other comments on a certain currently-displayed piece of art, and enable users to view another random image. Furthermore, it updated the online user lists and send notification when new users join or leave. Through this project, my goal was to enable the experience of a virtual exhibition, facilitate conversations about paintings, allow users to be creative with art games, or simply use it as a regular chatroom.
